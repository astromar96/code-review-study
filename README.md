# Code Review — Study App

Interactive reader for [code-review-study-guide.md](../code-review-study-guide.md).
Navigate by section, mark sections as studied, resume where you left off.
Progress is stored in `localStorage`.

## Run

```bash
npm install
npm run dev
```

Then open http://localhost:5173.

## Keyboard shortcuts

- `j` / `→` — next section
- `k` / `←` — previous section
- `x` — toggle studied on the current section
- `g` — jump to first section
- `G` — jump to last section

## Build

```bash
npm run build
npm run preview
```
